# Meeting Site

Веб-приложение для организации мероприятий.

## Для чего нужен

Meeting Site помогает пользователям создавать и отслеживать мероприятия. Вместо чатов и таблиц — один сайт, где можно зарегистрироваться, создать событие и посмотреть список всех мероприятий.

## Что умеет

- Регистрация и вход в систему
- Создание мероприятий (название, описание дата, время)
- Просмотр списка всех мероприятий
- Удаление своих мероприятий
- Защита от неавторизованного доступа

## Как запустить

1. Установите зависимости:

```bash
pip install -r requirements.txt
```
2. Запустите приложение

```bash
python meetschedule.py
```

3. Откройте в браузере

```
http://127.0.0.1:5000
```

## Технологии

- Python — язык программирования
- Flask — веб-фреймворк
- SQLite — база данных
- Bootstrap — CSS-фреймворк для интерфейса
